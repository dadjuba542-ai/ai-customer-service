## 已完成功能

### 1. 核心对话系统
- Coze API 对接，四种咨询类型（产品咨询/使用答疑/朋友圈帮写/口播文案帮写）
- 流式逐字回复、打字中动画
- 无需登录即可使用（匿名用户），带 token 可关联历史

### 2. 首页布局
- 顶部问候（三行：你好👋 / 我是AI宝儿 / 今天想让我为你做点什么）+ 3D 浮动动画头像（150×150px，紫色辉光 + 悬浮动画 + 透视阴影）
- 快捷功能四宫格（智能体入口，毛玻璃卡片）
- 热门问题标签云
- 最新资讯卡片列表（毛玻璃风格）
- 底部导航栏：左右两侧导航项 + 中央紫色渐变魔法按钮（带向外扩散辉光动画）
- 全部主卡片采用毛玻璃效果（`backdrop-filter: blur(10px)`）
- 主色调：`#8B5CF6`（紫色），圆角系统统一为 24px

### 3. 独立管理后台 (`/admin`)
- 数据看板：统计卡片 + 对话趋势图（Chart.js）+ 咨询类型分布（环形图）+ 最近对话 + 热门提问词云
- 日期筛选：支持近7天/30天/90天/全部/自定义，联动全部看板数据，趋势图默认30天上限90天
- 热门提问词云：字号按提问用户数比例缩放，品牌色板随机着色
- CSV 导出：根据当前日期筛选范围导出周报（统计概览/对话趋势/咨询类型/热门提问）
- 资讯管理：增删改查 + Quill.js 富文本编辑器 + 封面图上传
- 智能体配置：增删改查 + 颜色/图标/机器人ID/头像上传
- 系统配置：Coze API Key 在线更新 + 屏蔽关键词管理
- 等待小贴士管理：卡片列表 + 拖拽排序 + 逐条增删 + 批量保存

### 4. 等待回答体验优化
- 发送消息后，聊天记录内插入等待气泡（模仿机器人消息样式）
- 上半部分：步骤单行轮播（10步，每3.5秒切换，Phosphor 图标，淡入动画）
- 下半部分：小贴士轮播（每10秒切换一条，淡入动画）
- Coze 返回后自动移除气泡，切换为打字机输出
- 步骤和小贴士内容通过管理后台配置，默认值在 `app.py` 的 `DEFAULT_STEPS` / `DEFAULT_TIPS`

### 5. 资讯管理（管理后台）
- 增删改查资讯
- Quill.js 富文本编辑器
- 封面图上传 + 正文内图片插入

### 6. 智能体系统（动态）
- `GET /api/agents` 公开接口，前端动态加载
- 管理后台可新建/编辑/删除智能体
- `routes/chat.py` 优先使用智能体自身的 `bot_id`

### 7. 热门问题
- `GET /api/history/hot-questions`
- 按 `user_message` + `COUNT(DISTINCT user_id)` 去重聚合
- 自动过滤屏蔽关键词
- 管理员可配置屏蔽词

### 8. Coze API Key 在线更新
- 管理员后台可粘贴更新，存入 `settings` 表，无需重启

### 9. 安全机制
- 管理员接口使用 `admin_required` 装饰器（JWT 验证）
- 配置文件 `ADMIN_USERNAME` 控制注册时自动成为管理员

## 设计决策

### 等待气泡
- 独立面板改为聊天记录内气泡，不遮挡输入框
- 每次只显示当前一步（轮播替换），不留历史列表
- 图标使用 Phosphor Icons（与整体风格一致），不用 emoji
- 步骤和小贴士内容可后台配置，保存在 `settings` 表
- 默认值在 `app.py` 常量中，数据库为空时自动 fallback

### 管理后台
- 独立页面 `/admin` 而非 overlay 弹窗，用 `render_template` 渲染
- 侧边栏三页：数据看板 / 资讯管理 / 智能体配置
- 数据看板所有接口支持 `start_date` / `end_date` 参数
- 词云代替热门问题表格，CSV 导出纯前端实现（Blob 下载）

### 头像
- 用户提供的卡通图片代替 CSS 绘制的机器人
- 3D 效果：CSS perspective + rotate 微倾斜 + 浮动动画 + 紫色辉光
- 尺寸 150×150px

### 视觉设计
- 主色 `#8B5CF6`（紫色），圆角 24px，柔和阴影
- 毛玻璃效果：卡片 `background: rgba(255,255,255,0.6); backdrop-filter: blur(10px)`
- 底部魔法按钮：紫色渐变 + 径向辉光扩散动画
- 备份文件：`styles.old.css`、`index.old.html`（回退用）
